Это тестовый файл для проверки системы загрузки.

Первое предложение содержит важную информацию. Второе предложение дополняет первое. Третье предложение завершает первый абзац.

Новый абзац начинается здесь. В нем также есть несколько предложений. Они должны быть разбиты на чанки.

Заключительный абзац содержит финальные мысли. Система должна правильно обработать этот текст. Все предложения должны быть сохранены в векторной базе данных.
